## about

BrightGame is a simple Telegram game bot.

### General (current) features

"Memory" game (with multiple difficulty levels) is avaliable to all users.

### Admin tools

Admin user has the ability to use /status command. Options:
1. Pause/resume bot
2. Stop bot
3. Receive bot logs
